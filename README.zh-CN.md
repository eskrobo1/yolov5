## <div align="center">文档</div>

请参阅 [YOLOv5 文档](https://docs.ultralytics.com/yolov5/) 以获取关于训练、测试和部署的完整文档。下面提供了快速入门示例。

<details open>
<summary>安装</summary>

克隆仓库并在 [**Python>=3.8.0**](https://www.python.org/) 环境中安装 [requirements.txt](https://github.com/ultralytics/yolov5/blob/master/requirements.txt)，包括 [**PyTorch>=1.8**](https://pytorch.org/get-started/locally/)。

```bash
git clone https://github.com/ultralytics/yolov5 # 克隆仓库
cd yolov5
pip install -r requirements.txt # 安装依赖
```

</details>

<details open>
<summary>推理</summary>

YOLOv5 [PyTorch Hub](https://docs.ultralytics.com/yolov5/tutorials/pytorch_hub_model_loading/) 推理。 [Models](https://github.com/ultralytics/yolov5/tree/master/models) 会自动从最新的 YOLOv5 [release](https://github.com/ultralytics/yolov5/releases) 下载。

```python
import torch

# 加载 YOLOv5 模型（可选项: yolov5n, yolov5s, yolov5m, yolov5l, yolov5x）
model = torch.hub.load("ultralytics/yolov5", "yolov5s")

# 输入源（URL、文件、PIL 图像、OpenCV、numpy 数组 或 列表）
img = "https://ultralytics.com/images/zidane.jpg"

# 执行推理（支持批处理、调整大小、归一化）
results = model(img)

# 处理结果（可选方法：.print(), .show(), .save(), .crop(), .pandas()）
results.print()
```

</details>

<details>
<summary>使用 detect.py 进行推理</summary>

`detect.py` 可在多种输入源上运行推理，自动从最新的 YOLOv5 [release](https://github.com/ultralytics/yolov5/releases) 下载 [models](https://github.com/ultralytics/yolov5/tree/master/models)，并将结果保存到 `runs/detect` 目录。

```bash
python detect.py --weights yolov5s.pt --source 0                              # 调用摄像头
python detect.py --weights yolov5s.pt --source img.jpg                        # 单张图片
python detect.py --weights yolov5s.pt --source vid.mp4                        # 视频文件
python detect.py --weights yolov5s.pt --source screen                         # 屏幕截图
python detect.py --weights yolov5s.pt --source path/                          # 目录
python detect.py --weights yolov5s.pt --source list.txt                       # 图片列表
python detect.py --weights yolov5s.pt --source list.streams                   # 流列表
python detect.py --weights yolov5s.pt --source 'path/*.jpg'                   # 通配符匹配
python detect.py --weights yolov5s.pt --source 'https://youtu.be/LNwODJXcvt4' # YouTube 视频
python detect.py --weights yolov5s.pt --source 'rtsp://example.com/media.mp4' # RTSP、RTMP、HTTP 流
```

</details>

<details>
<summary>训练</summary>

以下命令重现 YOLOv5 [COCO](https://github.com/ultralytics/yolov5/blob/master/data/scripts/get_coco.sh) 结果。 [Models](https://github.com/ultralytics/yolov5/tree/master/models) 和 [datasets](https://github.com/ultralytics/yolov5/tree/master/data) 会自动从最新的 YOLOv5 [release](https://github.com/ultralytics/yolov5/releases) 下载。 在 V100 GPU 上，YOLOv5n/s/m/l/x 的训练时间分别为 1/2/4/6/8 天（[多 GPU](https://docs.ultralytics.com/yolov5/tutorials/multi_gpu_training/) 可大幅加速）。请使用最大的 `--batch-size`，或者传入 `--batch-size -1` 启用 YOLOv5 的 [AutoBatch](https://github.com/ultralytics/yolov5/pull/5092)。下面的批量大小以 V100-16GB 为准。

```bash
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5n.yaml --batch-size 128
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5s.yaml --batch-size 64
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5m.yaml --batch-size 40
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5l.yaml --batch-size 24
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5x.yaml --batch-size 16
```
